## Установка зависимостей

### Шаг 1 — Создать виртуальное окружение

Виртуальное окружение изолирует библиотеки проекта от системного Python.  
Открыть терминал в папке проекта и выполнить:

```powershell
python -m venv .venv
```

Будет создана папка `.venv/` рядом с файлами проекта.

### Шаг 2 — Активировать окружение

```powershell
.\.venv\Scripts\Activate.ps1
```

После активации в начале строки терминала появится `(.venv)`.

> Если PowerShell запрещает выполнение скриптов, выполните один раз:
> ```powershell
> Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
> ```

### Шаг 3 — Установить библиотеки

```powershell
pip install -r requirements.txt
```

Устанавливаются:
| Библиотека | Назначение |
|---|---|
| `numpy` | Матричные вычисления, генерация случайных данных |
| `scipy` | Венгерский алгоритм (`linear_sum_assignment`) |
| `matplotlib` | Графики в Jupyter-ноутбуке |
| `jupyter` / `notebook` | Запуск `strategy_experiment.ipynb` |

### Шаг 4 — Запустить GUI

```powershell
python assignment_gui.py
```

---

## Удаление окружения

Если окружение больше не нужно — просто удалите папку `.venv`:

```powershell
Remove-Item -Recurse -Force .venv
```

Файлы проекта не затрагиваются. Чтобы начать заново — повторите шаги 1–3.

## Руководство по GUI

### Общий вид

```
┌─────────────────────────────────────────────────────────────────┐
│  Верхняя панель: параметры, кнопки действий                     │
├──────────────────────┬──────────────────────────────────────────┤
│  Левая панель        │  Правая панель                           │
│  • Матрица A (ввод)  │  • Сетка назначений (визуализация)       │
│  • Вектор χ (ввод)   │  • Результаты сравнения                  │
│  • Список эвристик   │                                          │
└──────────────────────┴──────────────────────────────────────────┘
```

Все разделители между панелями **перетаскиваются мышью** — можно изменить
размер любой области под свой экран.

---

### Верхняя панель

| Элемент | Описание |
|---|---|
| **n =** | Размер матрицы (количество групп и периодов). Диапазон 2–12 |
| **A: ↓ убыв / ↑ возр / случайн** | Режим генерации строк матрицы при нажатии «Случайно» |
| **a ∈ [lo – hi]** | Диапазон случайных значений элементов матрицы A (должны быть > 0) |
| **χ ∈ (lo – hi)** | Диапазон случайных значений χᵢ (строго между 0 и 1) |
| **Случайно** | Заполнить матрицу и χ случайными значениями в заданных диапазонах |
| **Открыть** | Загрузить задачу из файла JSON или CSV |
| **Сохранить** | Сохранить текущую матрицу A и вектор χ в файл |
| **Решить** | Запустить выбранные эвристики и показать результаты |
| **Сбросить назначение** | Убрать все ручные назначения |

---

### Левая панель — ввод данных

#### Матрица A

Таблица ввода `n × n`. Строки — группы активов (`w1, w2, …`),  
столбцы — периоды времени (`t1, t2, …`).  
Значения можно менять вручную или загрузить из файла.

#### Вектор χ

Строка из n полей. Каждый `χᵢ` — коэффициент уязвимости группы `i`.  
Чем **меньше** χᵢ, тем группа уязвимее (больше теряет без защиты).  
Допустимые значения: строго от 0 до 1 (не включая границы).

#### Список эвристик

Галочками выбираются алгоритмы, которые будут запущены при нажатии **▶ Решить**.

| Параметр | Описание |
|---|---|
| **Моделей в ансамбле** | Количество агентов для `Ансамбль-B` и `Ансамбль-χ`. Больше → точнее, медленнее |
| **Фикс. seed** | Включить — ансамбль всегда даёт одинаковый результат (воспроизводимо). Выключить — случайный каждый раз |

---

### Правая панель — сетка назначений

#### Переключатель «матрицу A / матрицу B»

- **Матрица A** — исходные значения потенциальной прибыли. Рядом появляется столбец **χᵢ**
- **Матрица B** — вычисленные бонусные прибыли (`bᵢⱼ = (1−χᵢ)·Σcᵢₜ`). По ней работают жадная и гибридные стратегии

#### Цвета ячеек матрицы

Фон каждой ячейки отражает **относительную величину значения** в матрице:

| Цвет | Значение |
|---|---|
| Тёмно-серый | Низкое значение (наименее выгодная ячейка) |
| Тёмно-янтарный | Среднее значение |
| Тёмно-зелёный | Высокое значение (наиболее выгодная ячейка) |

Это помогает визуально найти «горячие» ячейки без чтения чисел.

#### Столбец χᵢ (виден при режиме «матрица A»)

| Цвет | Значение χᵢ |
|---|---|
| Тёмно-красный | Малое χᵢ — группа **уязвима** (большие потери без защиты) |
| Тёмно-зелёный | Большое χᵢ — группа **устойчива** (меньше теряет без защиты) |

#### Состояния ячеек

| Вид ячейки | Что означает |
|---|---|
| Обычная (цветной фон) | Ячейка доступна для назначения |
| **Синяя рамка + галочка ✓** | Выбрана вручную пользователем |
| Тёмная + зачёркнутое число | Недоступна: строка или столбец уже назначены |
| Цветные точки сверху | Какие эвристики выбрали эту ячейку (см. легенду внизу сетки) |

#### Ручное назначение

Кликните по ячейке — она будет выбрана (синяя рамка, галочка).  
Повторный клик на ту же ячейку — снять назначение.  
Выбор в том же столбце или строке автоматически отменяет предыдущее.  
Кнопка **↺ Сбросить** — убрать всё.

Когда все n групп назначены, в панели результатов появится строка **«👤 Ваше»** с прибылью.

#### Цветные точки эвристик

После нажатия **▶ Решить** в каждой ячейке, которую выбрала эвристика,
появляется цветная точка. Цвет — по легенде внизу сетки:

| Цвет точки | Эвристика |
|---|---|
| 🔴 Красная | Жадная |
| 🔵 Голубая | χ-стратегия |
| 🟢 Зелёная | Венгерский |
| 🟠 Оранжевая | Гибридная-1 |
| 🟣 Фиолетовая | Гибридная-2 |
| 🟧 Тёмно-оранжевая | Ансамбль-B |
| 🩵 Бирюзовая | Ансамбль-χ |

### Панель результатов

Таблица отсортирована по убыванию прибыли. Лучший результат выделен зелёным.

| Колонка | Описание |
|---|---|
| Цветная полоска | Цвет соответствующей эвристики |
| Метод | Название стратегии |
| Прибыль | Значение целевой функции F(σ) |
| Δ% vs Венг. | Отклонение от венгерского алгоритма в процентах. 0.00% = оптимум |
| Расписание | `t1→w2   t2→w5  …` — какая группа назначена в какой период |

### Форматы файлов

#### JSON
```json
{
  "A": [
    [7.2, 5.1, 3.0],
    [4.8, 6.3, 2.9],
    [9.1, 4.0, 1.7]
  ],
  "chi": [0.45, 0.62, 0.38]
}
```

#### CSV
```
0.45,0.62,0.38
7.2,5.1,3.0
4.8,6.3,2.9
9.1,4.0,1.7
```
Первая строка — вектор χ. Остальные строки — матрица A (по одной строке на группу).
