# GUI — Задача о назначениях

Интерактивное приложение для ручного и автоматического решения
модифицированной задачи о назначениях с помощью эвристических методов.

---

## Требования

- Python 3.9 или новее
- Библиотеки: `numpy`, `scipy`
- `tkinter` — входит в стандартную поставку CPython (устанавливать отдельно не нужно)

---

## Установка окружения

### 1. Создать виртуальное окружение

```bash
python -m venv venv
```

### 2. Активировать окружение

**Windows:**
```bash
venv\Scripts\activate
```

**macOS / Linux:**
```bash
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python assignment_gui.py
```

---

## Удаление окружения

Просто удалите папку `venv`:

**Windows:**
```bash
rmdir /s /q venv
```

**macOS / Linux:**
```bash
rm -rf venv
```

## Инструкция по приложению

### Верхняя панель

| Элемент | Описание |
|---|---|
| **n =** | Размерность задачи (матрица n×n) |
| **C: убыв / возр / случайн** | Режим генерации строк матрицы C |
| **c ∈ [lo – hi]** | Диапазон случайных значений матрицы C |
| **χ ∈ (lo – hi)** | Диапазон случайных значений вектора χ (строго между 0 и 1) |
| **Случайно** | Заполнить матрицу и χ случайными числами |
| **Открыть** | Загрузить задачу из файла `.json` или `.csv` |
| **Сохранить** | Сохранить текущую задачу в файл |
| **Решить** | Запустить выбранные эвристики |
| **Сбросить назначение** | Очистить ручное назначение |

---

### Левая панель — ввод данных

**Матрица C (cᵢⱼ)** — таблица ввода потенциальных прибылей.
Строки `w1..wn` — группы активов (ГИА), столбцы `t1..tn` — периоды времени.

**Коэф. уязвимости χᵢ** — вектор коэффициентов (0 < χ < 1) для каждой ГИА.
Чем меньше χ, тем уязвимее группа при отсутствии защиты.

**Эвристики для сравнения** — выберите нужные методы галочками перед запуском.

- **Выбрать все / Отменить все** — быстрый выбор.
- **Моделей в ансамбле** — количество агентов `m` для методов *Ансамбль-B* и *Ансамбль-χ*.
- **Фикс. seed** — при включении ансамбли дают воспроизводимый результат.

---

### Центральная панель — сетка назначений

Отображает матрицу в выбранном виде:

| Переключатель | Что показывает |
|---|---|
| **матрицу G** | Бонусная матрица `gᵢⱼ = (1−χᵢ)·Σₜ≥ⱼ cᵢₜ` |
| **матрицу D** | Критерий жадного выбора `dᵢⱼ = (1−χᵢ)·cᵢⱼ` |
| **матрицу C** | Исходная матрица прибылей (+ столбец χᵢ справа) |

**Цвет ячеек** — тёмно-серый (низкое значение) → янтарный → зелёный (высокое).

**Ручное назначение:** кликните на ячейку `(i, j)` — группа `i` будет назначена
на период `j`. Повторный клик снимает назначение.
Назначенные ячейки подсвечены синим, прочие в той же строке/столбце затемнены.

**Цветные точки** в верхней части ячейки — какие эвристики выбрали это назначение.

---

### Правая нижняя панель — результаты

После нажатия **▶ Решить** появляется таблица:

| Колонка | Описание |
|---|---|
| Цветная полоска | Цвет эвристики |
| Метод | Название (для ансамблей указывается `m`) |
| Прибыль | Значение целевой функции S₁(σ) |
| Δ% vs Венг. | Отклонение от оптимума (Венгерский = 0%) |
| Расписание | `t1→w3  t2→w1  …` |

Записи отсортированы по убыванию прибыли. Лучший результат выделен зелёным.

## Форматы файлов

### JSON

```json
{
  "C": [
    [8.2, 5.1, 3.0],
    [7.5, 8.8, 4.2],
    [6.0, 4.3, 9.1]
  ],
  "chi": [0.3, 0.5, 0.4]
}
```

### CSV

Первая строка — вектор χ, остальные строки — строки матрицы C:

```
0.3,0.5,0.4
8.2,5.1,3.0
7.5,8.8,4.2
6.0,4.3,9.1
```
